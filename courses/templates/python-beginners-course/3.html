Now that you have installed python on your PC, tt's time to get started with coding.

Python IDLE

Python IDLE(Integrated Development Environment) was already installed on your PC when you installed Python. It is the tool that we are going to use for testing small lines of code and learning the basics of Pyhton. After we finish with the basics we will install a more robust Code Editor(IDE) for working with more lines of code for doing exercises, projects(gonna be really exciting) on our way. For now stick with the Python IDLE.

Opening Python IDLE

Open 'Command Prompt' if you are on Windows by searching it in the start menu. Now enter python and hit 'enter'.

You will see something similar to this in your windows terminal too. As you can see in the image above I have installed Python 3.8.1. Here 8.1 is a subversion of Python3.

For Linux & MacOS users, as shown in the previous guide, you can open up the python IDLE by just opening in the 'Terminal' and typing in python3 and hitting enter. If you face any issues, please ask on the forum.


Great! Now we've got our IDLE running on your PC!

A..B..C..D...

Since we have the python IDLE opened you can start writing your first bits of python code!
The '>>>' sign you see is called a prompt. It means that the IDLE is ready to recieve codes to run. So let's test the IDLE prompt.

Type your name. for eg: "Hi I am James" with the quotes(either double or single) and hit enter.

As you can see the IDLE returned 'Hi I am James'.

Sweat!! You just ran your first lines of python code.

This might not be really exciting right now, but you should be proud now that you :

  • Installed Python3 on your PC
  • Opened the IDLE in a terminal.
  • Ran a Python code written by yourself!

Now try typing something else in the new prompt but remember to put quotes(try without quotes to see what would happen.) More explanation on that in the next chapter!

On the next page we will work with numbers and operations. Good Luck and see you there.
