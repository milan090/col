Installing Python3

There are 2 popularly know python versions : Python2 and Python3. In this course we will be using Python3 because of it being widely adapted now in Machine Learning and Web development frameworks.

Windows

Personally I use Windows(most) and Ubuntu(sorry MacOS users) so most of the course will be taught on Windows. But don't worry other than the installation part 99% of everything I do will look exactly the same no matter Linux or MacOS because of Python's portability accross different platforms and OS's.

  1. Download the latest version of Python here
  2. From your downloads folder, run the setup by double clicking on it.
  3. Check the option 'Add Python to PATH'.
  4. Choose Install Now
  5. Great you just installed python3 on your PC!

Linux and MacOS

Linux and MacOS already has python2 pre-installed in them, but since we need python3 we need to install those.

  • Linux

    Open terminal by pressing Ctrl + Alt + T on your keyboard.
    Now enter the below code in terminal.
    sudo apt-get install python3
    It will ask you to enter your password. Enter it and press Enter. Now it may ask you 'whether you want to install python3 or not', enter 'y' for yes and hit Enter.
    Yay you installed python3 on your linux.


    Enter 'python3' and hit enter to see if everything went right. If you are facing any problems ask on the forum.

  • MacOS

    Open terminal. Copy paste the following code into terminal and hit enter.
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    Now type :
    brew install python3
    You have successully installed python3 on your MacOS
    To check whether everything went right you can type 'python3' without codes and hit enter. If you are facing any problems ask on the forum.
